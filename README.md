# OpenGLCourseGuidance
&emsp;&emsp;北京交通大学2020年春季学期计算机学院计算机图形学课程指导教程

&emsp;&emsp;教程同时注重理论和实践，既尽可能地涉及到计算机图形学的算法原理，包括坐标系变换公式的推导方法，OpenGL图形库的机制等，同时也注重实践，配有充足的代码并对C++编程技巧有讲解。希望大家图形学都要拿个A鸭。
